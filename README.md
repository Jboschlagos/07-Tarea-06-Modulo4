Ejercicio Práctico - Funciones Callback en JavaScript

Funciones Callback en JavaScript (Validación y Cálculo)
Introducción / Objetivo
En esta actividad pondrás en práctica el uso de funciones callback en JavaScript, aplicando validaciones, cálculos matemáticos y manejo de asincronía mediante temporizadores. El objetivo es comprender cómo las funciones pueden ejecutarse como argumentos y responder a diferentes escenarios (éxito o error).
